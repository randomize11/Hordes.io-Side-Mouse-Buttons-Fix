# Hordes.io-Side-Mouse-Buttons-Fix
Forces standard side mouse buttons (m3/m4) to always apply on in-game canvas.

To use the side mouse buttons in Hordes.io, use "m3"/"m4" as an in-game key binding. "m3" refers to the lower side button "back", and "m4" refers to the upper side button "forwards".
This mod does not work on Firefox or any Gecko-based browsers. It works in Chrome. Other Chromium-based browsers have not been tested.
This mod only works with the standard 2 side buttons and does not support additional side buttons.
